# Simple Python Calculator 🧮

This is my first Python project! A simple command-line calculator that performs basic arithmetic operations.

## Features
* **Addition, Subtraction, Multiplication, and Division.**
* **Input Validation:** Checks if the operation exists.
* **Error Handling:** Prevents crashing when dividing by zero.
* **Looping:** Allows multiple calculations without restarting the program.
