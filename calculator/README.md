# Simple Python Calculator 🧮

A beginner-friendly command-line calculator built with Python to perform basic arithmetic operations.

---

## ⚙️ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division  
- 🔁 Continuous calculations without restarting the program  
- 🚫 Basic error handling (e.g., division by zero)
